.. tabs-cloud-providers::

   .. tab::
      :tabid: aws

      An AWS tab from the pre-configured Cloud Providers tab set.
      Note the lack of a user-facing tab name -- Snooty determines the
      name from the ``tabid`` alone.
   
   .. tab::
      :tabid: azure

      An Azure tab from the pre-configured Cloud Providers tab set.
      Note the lack of a user-facing tab name -- Snooty determines the
      name from the ``tabid`` alone.

   .. tab::
      :tabid: gcp

      A GCP tab from the pre-configured Cloud Providers tab set.
      Note the lack of a user-facing tab name -- Snooty determines the
      name from the ``tabid`` alone.

Because both tables use the same ``tabid`` values, switching between
tabs on either table also switches between tabs on the other table.
